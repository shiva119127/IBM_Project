# LangLearn

**LangLearn** is a modern, interactive language learning web application that helps users expand their vocabulary efficiently using flashcards, pronunciation support, quizzes, and progress tracking.

## Features

- **Interactive Flashcards:** Learn new words and their meanings in a visually appealing format.  
- **Pronunciation Support:** Listen to native-like pronunciation using the Web Speech API.  
- **Quiz Mode:** Test your knowledge and reinforce memory with instant feedback.  
- **Progress Tracking:** Keep track of your learning progress with scores updated in real time.  
- **Responsive Design:** Works seamlessly on desktop and mobile devices.
